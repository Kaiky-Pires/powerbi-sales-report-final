# Relatório Executivo de Vendas (Sales Report) - Power BI

# 1. Objetivo do Projeto
Este projeto final consolida os conhecimentos de Business Intelligence para a criação de um Relatório de Vendas robusto e intuitivo. O objetivo principal é fornecer uma visão clara sobre o desempenho comercial, permitindo a análise de tendências, faturamento e metas atingidas.

# 2. Composição do Relatório
O planejamento deste relatório foi estruturado para apresentar as seguintes visões:
Visão Geral (Overview): KPIs de faturamento total, ticket médio e volume de vendas. Uso de gráficos de área para demonstrar a evolução temporal das vendas ao longo dos meses.

Análise por Produto/Categoria: Gráficos de barras comparativos para identificar os produtos de maior giro e os que possuem maior margem de contribuição.

Desempenho Regional: Integração de mapas dinâmicos para visualização da distribuição geográfica das vendas, facilitando a identificação de mercados subexplorados.

# 3. Recursos Técnicos Aplicados
Interatividade Avançada: Planejamento de filtros (slicers) sincronizados entre as páginas para garantir uma navegação consistente.

Design e UX: Utilização de uma paleta de cores sóbria e layout profissional, focando na redução da carga cognitiva do usuário final.

Dicas de Ferramenta (Tooltips): Configuração de visuais de detalhes que aparecem ao passar o mouse sobre os dados, fornecendo informações extras sem poluir o visual principal.

# 4. Justificativa de Entrega
Devido às limitações técnicas de ambiente (ausência de Power BI Desktop para Windows), a entrega foca na documentação técnica de Storytelling e Design de Dados. O repositório descreve a arquitetura final do relatório, demonstrando a capacidade de transformar requisitos de negócio em um dashboard executivo eficiente.
